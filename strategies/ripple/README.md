# 🌊 涟漪策略 (Ripple Strategy)

> 自适应牛熊的越跌越买策略

## 逻辑

用 **MA60** 判断市场趋势：
- 🐂 **牛市** (价格 > MA60)：越跌越买(×3 递增) + 追涨 $5 + 每天卖 0.5%
- 🐻 **熊市** (价格 < MA60)：保守买入(×2 递增) + 不追涨 + 每天卖 1%

## 回测

- 3 年收益: **+57.41%** (BTC +144.91%)
- 最大回撤: **19.89%** (BTC 51.16%)
- 熊市(1年): **-20.0%** (BTC -40.18%)

## 运行

```bash
node strategy.js
node strategy.js --live
```
